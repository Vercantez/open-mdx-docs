Code blocks are highlighted at build time with [Shiki](https://shiki.style). Both light and dark themes are embedded as CSS variables, so switching themes needs no JavaScript.

## Code groups

Group related snippets with `CodeGroup`. Children are fenced code blocks; the first word of each fence language line is used as the tab title.

<CodeGroup>
```ts server.ts
export function greet(name: string) {
  return `Hello, ${name}`;
}
```

```ts client.ts
import { greet } from './server';

console.log(greet('world'));
```
</CodeGroup>
